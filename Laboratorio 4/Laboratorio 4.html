Lista

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    • Javascript no necesita un compilador, sino que basta con que el navegador lo interprete.
    • Java es un lenguaje orientado a objetos, mientras que Javascript no necesita declarar clases.
    • Javascript está limitado a páginas web, mientras que Java es de propósito general.
    • Javascrip no está fuertemente tipado (cualquier variable cambia cuando sea necesario). Java debe declarar su tipo, el cual no cambia de forma automática.
  2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
    • now(): regresa el número de milisegundos desde la medianoche de enero 1 de 1970.
    • parse(): regresa la cantidad de milisegundos entre la medianoche de enero 1 de 1970 y la fecha introducida como parámetro.
    • toString(): convierte una fecha en una cadena.
    • toUTCString(): crea una cadena con la fecha del Tiempo Universal Coordinado (UTC).
    • getUTCMilliseconds(): regresa la cantidad de milisegundos acorde al UTC, el rango es entre 0 y 999. Si se usaran minutos, sería entre 0 y 59. De ser días, sería entre 0 y 6.
  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)?
    • toString(): convierte el arreglo en una cadena separada por comas.
    • join(): permite escoger el separador con el que se convierte el arreglo en cadena (por ejemplo: " - ").
    • pop(): remueve el último elemento de una lista y retorna el valor del que fue borrado.
    • push(): añade un nuevo elemento al inicio de la lista. pop("Casa").
    • slice(): crea una nueva lista nueva desde la posición indicada en el parámetro. Cabe destacar que las posiciones de la lista empiezan desde 0.
  4. ¿Cómo se declara una variable con alcance local dentro de una función?

    Dentro de la misma función se debe declarar la variable. Es decir, dentro de los corchetes.

  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    Más de una función podrá modificarla. Además, no se reinician cada vez que se llama una función.

  6. ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web? includes() busca los caracteres que están incluidos en una cadena. Es un método muy útil en una base de datos.
