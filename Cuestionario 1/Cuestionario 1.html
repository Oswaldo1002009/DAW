Cuestionario 1

¿Qué es entonces una base de datos - DB?

Es un conjunto de información agrupada por sus tipos de atributos de manera organizada. Para las ciencias computacionales, estos datos pueden ser almacenados y consultados de forma directa y rápida.[1]

¿En qué casos es conveniente usar bases de datos?

  1. Cuando existen grandes cantidades de información.
  2. Cuando hay una desorganización que puede hacer difíciles las consultas o perder datos.
  3. Cuando se requieren hacer muchos cálculos o análisis complejos con los datos.
  4. Cuando esta información debe ser accedida simultáneamente por varias personas.
  5. Cuando los datos cambian constantemente. También aplica cuando la base de datos se expande y necesita más cosas.[2]

¿Qué es un sistema de gestión de base de datos - DBMS?

Es un programa (o conjunto de) que permite crear, actualizar y administrar una base de datos. Esto es gracias a herramientas para añadir, eliminar, modificar o consultar información. En las DBMS existen herramientas de consulta y niveles de acceso para los tipos de consultores (como administrador, empleado, etc.). Además tienen sistemas de seguridad para proteger los datos.[1]

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

  1. Administrar el acceso a los datos.
  2. Eficientar la administración de la base de datos.
  3. Eficientar la generación de reportes.
  4. Mantener la integridad de la información.
  5. Realizar copias de seguridad de forma rápida.
  6. Recuperar los datos tras una falla.[2]

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

Durante mi servicio social en preparatoria colaboré en una casa hogar para niñas. El lugar tenía un enorme almacén en donde guardaban todas las donaciones. Aunque las necesidades de la casa hogar eran muchas, no evitaba que tuvieran excesos de donaciones. El equipo en el que trabajé tuvo la tarea de inventarear cada una de las cosas que había en el almacén con el objetivo de saber qué donativos necesitaban y cuáles ya no. Aunque una base de datos habría sido muy conveniente, existía el problema de que, hasta donde recuerdo, no había nadie que pudiera gestionarla. Es posible que ni siquiera tuvieran computadoras allí.
