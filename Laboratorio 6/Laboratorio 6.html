Lista

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    • Javascript no necesita un compilador, sino que basta con que el navegador lo interprete.
    • Java es un lenguaje orientado a objetos, mientras que Javascript no necesita declarar clases.
    • Javascript está limitado a páginas web, mientras que Java es de propósito general.
    • Javascrip no está fuertemente tipado (cualquier variable cambia cuando sea necesario). Java debe declarar su tipo, el cual no cambia de forma automática.
